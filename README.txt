# Redimensionador de Imagens

Bem-vindo ao Redimensionador de Imagens! Este software permite redimensionar imagens de forma fácil e rápida.

## Instalação

### Requisitos
- Python 3.12 (recomendado, caso queira usar o main.py)
- Bibliotecas necessárias (instaladas automaticamente durante o processo de instalação, caso usado o executavel)

## Opções Avançadas

- **Qualidade:** A qualidade da imagem pode ser ajustada no código-fonte.
- **Algoritmo de Redimensionamento:** O algoritmo padrão é Lanczos, mas você pode alterar no código-fonte se desejar.
